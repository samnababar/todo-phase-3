# Task Search Skill

## Purpose
Quickly find tasks by keyword.

## Process
1. Search input in toolbar with magnifying icon
2. Debounced real-time filtering
3. Highlight matches in results
4. Search name + description fields

## Examples
Input: bg-[#1A1A1A] border-[#262626] focus:border-[#8B5CF6]

## Guidelines
- Use obsidian-theme-skill
- Client-side for speed
- Combine with other filters
