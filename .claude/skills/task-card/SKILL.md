# Task Card Skill

## Purpose
Beautiful, information-rich task display.

## Process
1. Card background: #1A1A1A
2. Title + description
3. Priority badge (violet intensity by level)
4. Status checkbox (emerald on complete)
5. Tags as violet pills
6. Hover: subtle lift + violet glow
7. Edit/delete icons on hover

## Examples
Priority high: bg-[#8B5CF6]/20 text-[#8B5CF6] badge
Tags: bg-[#8B5CF6]/10 text-[#A78BFA] rounded-full px-3 py-1

## Guidelines
- Use obsidian-theme-skill
- Truncate long text gracefully
- Accessible focus states
