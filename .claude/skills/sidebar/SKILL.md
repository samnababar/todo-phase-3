# Sidebar Skill

## Purpose
Persistent navigation with user context.

## Process
1. Fixed dark sidebar (#0F0F0F)
2. Top: User avatar + username
3. Middle: Navigation links (Dashboard, Tasks, Settings)
4. Bottom: Logout slider/button
5. Violet active state indicator

## Examples
Active link: border-l-4 border-[#8B5CF6] bg-[#1A1A1A]

## Guidelines
- Collapsible on mobile
- Use obsidian-theme-skill
- Smooth hover effects
